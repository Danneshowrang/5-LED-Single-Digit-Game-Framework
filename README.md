# 5-LED-Single-Digit-Game-Framework
A game framework comprised of 5 LEDs, and single-digit display, and a single button. Extremely simple, but that's the point!

# About
The framework is made up of 5 LEDS, a single-digit display, and a single button. Although this may seem simple, this bare-bones framework can be used to play a large variety of games, such as the reaction speed test I programmed for demonstration. Other examples are a sequential memory game, hot potato, dice roller, and shooting stars.
